🧹 Clear Old Cache Data

Why do this?
After implementing user authentication, old vocabulary and flashcard data stored in localStorage needs to be cleared. Each user now has their own separate data.

This will remove the following old localStorage keys:

  • vocabulary (old global vocabulary)
  • flashcards (old global flashcards)

Note: Your user-specific data (vocabulary_user-XXX, flashcards_user-XXX) will NOT be deleted.
